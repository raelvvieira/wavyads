# SKILL: Google Ads Brasileiro — Criação de Campanhas de Alta Conversão

## QUANDO USAR ESTA SKILL
Usar sempre que o usuário pedir para criar, estruturar ou revisar campanhas de Google Ads (Search ou Performance Max) para clientes no Brasil, especialmente pequenos negócios locais. Também usar ao criar grupos de anúncios, palavras-chave, títulos, descrições, sitelinks, callouts ou snippets em português brasileiro.

---

## FILOSOFIA BASE — O MÉTODO

### Princípio #1: Message Match
Cada grupo de anúncios fala de UM só serviço. Quem busca "limpeza de sofá" vê anúncio sobre sofá. Quem busca "implante dentário" vê anúncio sobre implante. Nunca misturar serviços no mesmo grupo.

**Impacto direto:**
- Quality Score médio: 8–10 (vs 4–6 de campanhas genéricas)
- CPC até 40% menor
- CTR 2–3x maior

### Princípio #2: Especificidade Geográfica
Toda campanha é por cidade. Campanha de Florianópolis ≠ Campanha de Porto Alegre. A cidade aparece nos títulos, descrições, palavras-chave e caminhos de exibição. O Google trata isso como sinal fortíssimo de relevância.

### Princípio #3: Títulos 1 e 2 Fixados
Os dois primeiros títulos são SEMPRE fixados (pinned). São os que o usuário mais vê. Devem conter: serviço + cidade.

### Princípio #4: CTA WhatsApp
O público brasileiro converte muito mais por WhatsApp do que por formulário. O CTA padrão é:
- "Agende pelo WhatsApp"
- "Fale no WhatsApp agora"
- "Orçamento grátis no WhatsApp"
- "Atendimento pelo WhatsApp"

### Princípio #5: Prova Social Sempre que Possível
Anos de experiência, número de avaliações, certificações, diferenciais concretos. Nunca usar adjetivos vagos sem embasamento.

## ESTRUTURA COMPLETA DE CAMPANHA

### Hierarquia
```
CAMPANHA (por cidade + nicho)
  └── GRUPO 1 — Serviço Principal
  └── GRUPO 2 — Serviço Secundário
  └── GRUPO 3 — Serviço Terciário
  └── GRUPO N — ...
```

### Nomenclatura de grupos
- Kitesurf Adventure: `Aulas de Kitesurf | Curso de Kitesurf | Wing Foil | Kitesurf Avançado | Kitetrip`
- Dentista: `Dentista Florianópolis | Estética Facial Floripa | Alinhadores Floripa | Clareamento Floripa`
- Limpeza domiciliar: `Limpeza de Sofá | Limpeza de Colchão | Limpeza de Tapete | Higienização Geral`

## OUTPUT COMPLETO POR GRUPO

Para cada grupo entregar na seguinte ordem:

### 1. CAMINHO DE EXIBIÇÃO (Display Path)
```
dominio.com.br › slug-do-servico › cidade
```
Exemplo: `kitesurfadventure.com.br › aulas-kitesurf › florianopolis`
Regra: nunca usar acentos nem espaços no slug. Cidade sempre sem acento no path.

### 2. TEXTO ÚNICO (máx 270 chars para PMax)
Parágrafo descritivo do serviço para geração de recursos pela IA do Google. Deve conter:
- Nome do negócio
- Serviço específico
- Cidade
- 2–3 diferenciais
- CTA WhatsApp

Exemplo:
```
Kitesurf Adventure oferece aulas de kitesurf em Florianópolis na Lagoa da
Conceição e Praia Daniela. Escola certificada IKO com 17 anos de experiência.
Equipamento incluso para iniciantes e avançados. Agende pelo WhatsApp.
```

### 3. TÍTULOS CURTOS (15 títulos, máx 30 chars cada)
- **Títulos 01 e 02: FIXADOS** — devem conter serviço + cidade
- Títulos 03–15: livres, explorando ângulos diferentes
- Nenhum com ponto, exclamação ou interrogação no final

**Ângulos a cobrir nos 15 títulos:**
| Ângulo | Exemplo |
|--------|---------|
| Serviço + Cidade (FIXAR) | `Dentista em Florianópolis` |
| Nome + Cidade (FIXAR) | `Dra. Mariane Botti Floripa` |
| Serviço + Local específico | `Kitesurf Lagoa Conceição` |
| Nível/público | `Kitesurf para Iniciantes` |
| Tempo de mercado | `17 Anos Ensinando Kitesurf` |
| Certificação | `Escola IKO Certificada` |
| Equipamento/estrutura | `Equipamento Incluso no Curso` |
| CTA WhatsApp | `Agende pelo WhatsApp Agora` |
| Orçamento | `Orçamento Grátis no WhatsApp` |
| Ação + serviço + cidade | `Aprenda Kitesurf em Floripa` |
| Benefício emocional | `Voe Sobre a Água em Floripa` |
| Variedade/modalidade | `Individual ou em Dupla` |
| Garantia/segurança | `Resultado no Mesmo Dia` |
| Reserva/urgência | `Reserve Seu Curso Hoje` |
| Nome do negócio | `Kitesurf Adventure Floripa` |

### 4. TÍTULOS LONGOS (5 títulos, máx 90 chars cada)
Mais narrativos, aparecem em Display/YouTube no PMax.
Estrutura: `[serviço em cidade] — [diferencial] | [CTA curto]`

Exemplos:
```

```

### 5. DESCRIÇÕES (5 descrições, máx 90 chars cada)
Aparecem no anúncio de busca abaixo dos títulos. Mais factuais que narrativas.
- Cada descrição = 1 benefício ou argumento concreto
- Sempre terminar com pontuação ou CTA
- Nunca usar newline dentro da descrição (quebra o contador de chars)

**Fórmulas que funcionam:**
```
[verbo + serviço + cidade]. [diferencial]. [CTA].
Aprenda kitesurf em Floripa. Escola IKO certificada. Equip. incluso.

[benefício emocional]. [prova social]. [CTA].
Voe sobre a água na Lagoa da Conceição. Instrutores certificados IKO.

[serviço] + [cidade] + [ação urgente].
Wing foil do zero com segurança em Florianópolis. Agende no WhatsApp!

[tempo/anos]. [modalidades disponíveis]. [cidade].
17 anos de escola. Curso individual ou em dupla em Floripa.

[nome negócio]: [serviço] para [público]. [cidade].
Kitesurf Adventure: wing foil para iniciantes e avançados em Floripa.
```

### 6. PALAVRAS-CHAVE (mín 10–15 termos por grupo)

**Usar os três tipos de correspondência:**
- `[exata]` — colchetes
- `"frase"` — aspas
- sem formatação — ampla

**Padrão de construção das keywords:**
```
[serviço + cidade]              → exata, maior precisão
"serviço + cidade"              → frase, variações
"serviço + a domicílio"         → frase, modificador de serviço
"serviço + cidade abreviada"    → frase, como "floripa", "poa"
[serviço + bairro/local]        → exata, hiperlocal
"melhor + serviço + cidade"     → frase, intenção de comparação
"aprender/fazer + serviço"      → frase, intenção informacional com alta conversão
serviço + cidade                → ampla, captura variações
serviço + bairro                → ampla, hiperlocal
```

## REGRAS TÉCNICAS CRÍTICAS

### Limites de caracteres
| Campo | Limite |
|-------|--------|
| Título curto | 30 chars |
| Título longo | 90 chars |
| Descrição | 90 chars |
| Sitelink (texto) | 25 chars |
| Sitelink (descrição 1 e 2) | 35 chars |
| Callout / Frase de destaque | 25 chars |
| Snippet estruturado | 25 chars |
| Texto único PMax | 270 chars |
| Caminho de exibição | 15 chars por nível |

### Erros comuns que DEVEM ser evitados
1. **Newline nas descrições** → quebra o contador de chars, entrega texto maior que o real
2. **Pontuação em callouts** → Google rejeita frases de destaque com `.`, `!` ou `?` no final
3. **Misturar cidades no mesmo grupo** → destrói o message match
4. **Não fixar títulos 1 e 2** → Google rotaciona e o usuário pode ver título genérico primeiro
5. **Acentos faltando nas keywords** → `florianopolis` ≠ `florianópolis` no Google Brasil
6. **Snippets acima de 25 chars** → Google trunca ou rejeita

## FORMATO DE ENTREGA

Sempre entregar como **texto plain no chat**, nunca como documento. Organizado exatamente nesta sequência:

1. Caminho de exibição
2. Texto único
3. Títulos curtos (tabela: nº | texto | chars)
4. Títulos longos (numerados)
5. Descrições (numeradas)
6. Palavras-chave (separadas por ·)
7. Frases de destaque (separadas por |)
8. Snippets estruturados (separados por |)
9. Sitelinks (blocos por sitelink)
10. Negativos (separados por ·)
11. Temas de pesquisa (um por linha, em bloco de código)
12. Público-alvo

Cada grupo entregue completo de uma vez. Não entregar parcialmente e perguntar se continua.
